# Chromium Docker Image With Latest Node JS and Puppeteer

## Get This Docker Image
```shell
docker pull mrofi/chromium:<TAG>
```

## Working Dir
This image use /app as working dir

## Contains
This image contains :
- Ubuntu 20:04
- Node version 16
- Puppeeter version 10
